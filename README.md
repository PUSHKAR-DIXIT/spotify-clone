# spotify-clone
Spotify Clone (HTML &amp; CSS) A responsive Spotify-inspired music player UI built using pure HTML and CSS. This project focuses on layout design, flexbox, grid, and modern CSS styling to recreate the look and feel of Spotify’s web interface.  Tech Used:  HTML5  CSS3
Spotify Clone – Frontend UI Project
This is a frontend Spotify clone created using HTML and CSS only.
The project replicates Spotify’s homepage UI with responsive design, clean layout, and modern styling techniques.
Built to strengthen core frontend skills without using any frameworks or JavaScript.

Key Features:

Spotify-like UI

Responsive layout

Clean & structured CSS
